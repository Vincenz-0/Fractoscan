# Fractoscan

---

## 🚀 Project Overview
A React-based project built using Vite.

---

## ⚙️ React + Vite Template Info

This template provides a minimal setup to get React working in Vite with HMR and ESLint rules.

### Plugins available:
- @vitejs/plugin-react → uses Babel (or oxc) for Fast Refresh  
- @vitejs/plugin-react-swc → uses SWC for Fast Refresh  

---

## 🧠 React Compiler
The React Compiler is not enabled by default due to performance impact.  
To enable it, refer to official React documentation.

---

## 🔍 ESLint & TypeScript
For production apps:
- Use TypeScript
- Enable type-aware linting (`typescript-eslint`)
